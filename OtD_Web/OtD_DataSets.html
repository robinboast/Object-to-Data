Data Sets

This page provides links to a number of data sets that will be used throughout the course for assignments. Students can use these for the group projects as well.

  • NY Times API records (16 Mb):

    This data set contains metadata for NY Times newspaper articles containing the word 'humanities', taken from the NY Times API, which provides access to the entire archive from 1850 onwards. The metadata records do not contain the full text of the articles, but a short description and a range of metadata.

    • The Guardian API records (169 Kb):

    This data set contains metadata for The Guardian newspaper articles containing the word 'humanities', taken from The Guardian API, which provides access to 1.7 million articles from 1999 onwards. The metadata records do not contain the full text of the articles, but only metadata.

    Amazon book reviews (24 Mb):

    This data set contains a sample of book reviews for 23 well-known books.

    Lost in Translation (2003) Amazon movie reviews (2 Mb):

    This data set contains a sample of movie reviews for the film Lost in Translation (2003).

    Amazon movie reviews (58 Mb):

    This data set contains a sample of movie reviews for 153 well-known movies.

    Amazon music reviews (17 Mb):

    This data set contains a sample of music album reviews for 40 well-known music albums.
